Timevora is an installable, offline-first productivity web app designed for students to track study time, build routines, and maintain consistency.  
It runs entirely in the browser with no backend and stores data locally for privacy and performance.

## Features

### Study Tracking
- Stopwatch-based session tracker
- Topic/subject labeling
- Pause and resume support
- Automatic logging and totals

### Timers
- Pomodoro-style study/break modes
- Preset durations
- Custom time entry

### Goals & Habits
- One-day goals
- Recurring goals
- Completion toggles
- Streak tracking

### Routine Matrix
- Time-structured daily planner
- Activity completion tracking

### Analytics & History
- Heatmap calendar archive
- Monthly totals
- Performance indicators
- Historical session logs

### Interface
- Responsive layout (mobile/tablet/desktop)
- Fullscreen focus mode
- Smooth navigation between screens
- Audio interaction feedback
- Live digital clock & date

---

## Progressive Web App Capabilities

- Installable on desktop and mobile

- Offline cold-start support
